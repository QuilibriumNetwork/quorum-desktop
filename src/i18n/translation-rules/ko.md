# Korean (ko) — translation rules

## Register: semi-formal (당신 form + 합니다 polite ending)

Korean has multiple registers — use the standard polite middle ground throughout.

### Pronouns / address
- "you" → 당신 (NOT 귀하 honorific, NOT 너 informal)
- "your" → 당신의 (NOT 귀하의, NOT 너의)
- Verbs end in `-합니다` / `-입니다` polite form

### Common failure modes
- Model drops to 귀하의 honorific for short labels ("Your Details", "Your Roles") while keeping 당신의 for longer text. Stay consistent.
- Model uses bare informal `너` form — never do this.

### Examples
- "Your account" → "당신의 계정"
- "Your Details" → "당신의 세부정보" (NOT "귀하의 세부정보")
- "You're offline" → "오프라인 상태입니다"

### Brand terms
Keep in English: Quorum, Quilibrium, passkey, YouTube. Use loanword form (transliterated) for: Space → 스페이스 (or 공간), Channel → 채널.

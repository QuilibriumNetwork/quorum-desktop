# Greek (el) — translation rules

## Register: INFORMAL (singular εσύ family)

Greek has both formal (plural Εσείς) and informal (singular εσύ) registers. **Use ONLY informal singular throughout.** This is the most common failure mode for Greek — the model often flips to formal mid-batch.

### Pronouns & verb endings
- "you" → εσύ (NOT Εσείς)
- "your" → σου (NOT σας)
- "you are" → είσαι (NOT είστε)
- "you have" → έχεις (NOT έχετε)
- "you send" → στέλνεις (NOT στέλνετε)
- Imperative: "wait" → περίμενε (NOT περιμένετε)

### Examples
- "Your account" → "ο λογαριασμός σου"
- "You're offline" → "Είσαι εκτός σύνδεσης"
- "Please wait" → "Παρακαλώ περίμενε"
- "You have new messages" → "Έχεις νέα μηνύματα"

### Brand terms
Keep these in English: Quorum, Quilibrium, passkey, YouTube

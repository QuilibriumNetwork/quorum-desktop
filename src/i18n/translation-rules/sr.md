# Serbian (sr) — translation rules

## Register: FORMAL (Vy/vaš)

Use formal address throughout. Common failure: model uses formal `Vy` for short labels then drops to informal `ti/tvoj` for longer sentences.

### Pronouns
- "you" → vi (NOT ti)
- "your" → vaš/vaša/vaše (NOT tvoj/tvoja/tvoje)
- "yourself" → vas/sebe (NOT te)
- Imperatives: "imagine" → Zamislite (NOT Zamisli); "keep" → Čuvajte (NOT Čuvaj); "you can" → Možete (NOT Možeš)

### Glossary (from prior audits)
- Mute = "Isključi zvuk"
- Unmute = "Ponovo uključi zvuk"
- Space = "Prostor"

### Examples
- "Your communities" → "Vaše zajednice"
- "You can download" → "Možete preuzeti"
- "Think of it as" → "Zamislite to kao"

### Brand terms
Keep in English: Quorum, Quilibrium, passkey, YouTube. Also TRANSLATE phrases like "YouTube Video Thumbnail" — only the brand name itself stays English, the rest gets translated.

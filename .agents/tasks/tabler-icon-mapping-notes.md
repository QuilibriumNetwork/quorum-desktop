## Usage Notes

### Animation Requirements

- **spinner** (IconLoader): Should be animated with CSS rotation
  ```css
  @keyframes spin {
    from { transform: rotate(0deg); }
    to { transform: rotate(360deg); }
  }
  .icon-spinner {
    animation: spin 1s linear infinite;
  }
  ```

### Filled vs. Outline Variants

Tabler Icons has both outline and filled versions for many icons:
- **circle**: IconCircle (outline) vs. IconCircleFilled (filled)
- **heart**: IconHeart (outline) vs. IconHeartFilled (filled)
- **star**: IconStar (outline) vs. IconStarFilled (filled)

If FontAwesome icons were using filled versions, update mapping to use Filled variants.

### Size Adjustments

Some Tabler icons may appear slightly larger or smaller than FontAwesome at the same pixel size. Common adjustments:
- If icon feels too small: increase size by 1-2px
- If icon feels too large: decrease size by 1-2px
- Test at all size breakpoints (xs through 5xl)

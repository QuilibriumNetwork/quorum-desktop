description: Load styling guidelines before CSS/SCSS work
---

Before writing or modifying CSS/SCSS, read the project's styling guidelines:

1. **`.agents/docs/styling-guidelines.md`** - Core rules: when to use Tailwind vs raw CSS, SCSS variables, color system, form fields, breakpoints
2. **`src/styles/_variables.scss`** - All SCSS variables ($s-4, $text-sm, $rounded-lg, etc.)
3. **`src/styles/_colors.scss`** - Theme colors, surfaces, accents

Follow these patterns throughout your CSS work.
